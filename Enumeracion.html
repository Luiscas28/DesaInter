Conceptos Importantes de HTML

  1. HTML - HyperText Markup Language, es el lenguaje de marcado utilizado para estructurar y presentar contenido en la web.
  2. Etiqueta - Un elemento que define la estructura y el contenido en un documento HTML, generalmente está formada por una etiqueta de apertura y una de cierre, como <h1> y </h1>.
  3. Elemento - Un conjunto de etiquetas HTML que contienen el contenido, atributos y el comportamiento de un documento, como <p> (párrafo).
  4. Enlace - Un elemento HTML que permite redirigir a los usuarios a otras páginas web o recursos, utilizando la etiqueta <a> junto con el atributo href.
  5. CSS - Cascading Style Sheets (Hojas de Estilo en Cascada), se utiliza para definir la presentación visual de un documento HTML, como colores, fuentes y diseño.
  6. HTML5 - La última versión de HTML, que incluye soporte para multimedia (audio y video), APIs de JavaScript, y una estructura semántica mejorada, entre otras funcionalidades.
