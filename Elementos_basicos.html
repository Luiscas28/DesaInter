Lista de elementos básicos

  • Etiqueta "html": marca el inicio y el final del documento HTML
  • Etiqueta "head>": se utiliza para incluir información adicional en el documento, como el título de la página o enlaces a hojas de estilo.
  • Etiqueta "body": marca el inicio y el final del contenido principal de la página web.
  • Etiqueta "p": se utiliza para crear párrafos de texto.
  • Etiqueta "img": se utiliza para insertar imágenes en una página web.
  • Etiqueta "a": se utiliza para crear enlaces a otras páginas web o recursos.
  • Etiqueta "div": se utiliza para agrupar otros elementos HTML en una sección lógica y separada de la página.
  • Etiqueta "span": se utiliza para aplicar estilos específicos a un fragmento de texto dentro de una página web.
  • "!DOCKTYPE":Es la declaración que debe ir al principio de un documento HTML para indicar la versión de HTML que se está utilizando (por ejemplo, HTML5)
  • "title":Descripción: Define el título de la página que aparece en la pestaña del navegador. Se coloca dentro de la etiqueta "head".
  • "Meta:Se utiliza para incluir metadatos como la codificación de caracteres, descripción de la página y palabras clave. También puede definir información importante para motores de búsqueda.
  • Etiqueta "h1" a "h6":Se utilizan para definir encabezados o títulos de diferentes niveles, desde el más importante (h1) hasta el menos importante (h6).
  • Etiqueta "ul" y "ol":Sirven para crear listas. "ul" crea una lista desordenada (con viñetas) y "ol" crea una lista ordenada (numerada).
  • Etiqueta "form":Define un formulario para recopilar datos del usuario, como nombres, correos electrónicos o contraseñas.
  • Etiqueta "input":Se utiliza para crear campos interactivos de entrada, como texto, contraseñas, botones, etc.
  • Etiquet "Button":Define un boton clicable
  • Etiquera "Script"Permite incluir o enlazar código JavaScript en la página.
  • Etiqueta"Link":Se utiliza para enlazar recursos externos, como hojas de estilo CSS.
